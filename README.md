# Exercícios de Algoritmo em Java Core

Este repositório serve como um compilado dos diversos **exercícios de algoritmo** que venho desenvolvendo no meu processo de aprendizado. O objetivo principal é aprimorar minhas habilidades em **lógica de programação e desenvolvimento de código** utilizando **Java core**.

Aqui você encontrará minhas soluções e abordagens para diferentes tipos de problemas algorítmicos.

---

## Conteúdo dos Exercícios

Cada exercício consiste em um arquivo `.java` com a implementação da solução.

Você pode encontrar a **lista completa e detalhada de todos os exercícios** realizados, com suas respectivas descrições e objetivos, no arquivo **[`lista-de-exercicios.md`](lista-de-exercicios.md)**.

Sinta-se à vontade para explorar os exemplos e acompanhar o meu progresso!
